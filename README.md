# 🔥 IP TRACE-X 🔥

---

## ⚡ ABOUT TOOL

IP TRACE-X হলো একটি Cyber Intelligence & Network Information Tool যেটা বিভিন্ন public network data analyze করতে সাহায্য করে।  
এই tool মূলত educational, research এবং security learning purpose এর জন্য তৈরি করা হয়েছে।

এখানে বিভিন্ন ধরনের IP intelligence, blacklist status, metadata analysis এবং system metrics check করা যায়।

---

## 🧩 MODULES

```bash
[01] YOUR IP SCAN (INTEL REPORT)
→ নিজের IP information ও network intelligence check

[02] TARGET IP TRACK
→ Target IP এর basic public information lookup

[03] NUMBER METADATA (GLOBAL)
→ International number metadata analysis

[04] IP BLACKLIST CHECK
→ IP blacklist database status verify

[05] SYSTEM INTERNAL METRICS
→ System/network related internal metrics display

[06] EXIT
→ Tool close
